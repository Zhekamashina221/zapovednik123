Диаграммы для раздела 2.1 дипломного проекта «Zapovednik»
==========================================================

Файлы PlantUML (вставить на https://plantuml.com/ или https://www.plantuml.com/plantuml/uml/):

  01-use-case.puml   → Рисунок 2.1 — Диаграмма вариантов использования
  02-activity.puml       → Рисунок 2.2 — Деятельность (карточка, избранное, маршрут)
  02-activity-app.puml   → Рисунок 2.2 (вариант 2) — Деятельность всего приложения
  03-class.puml      → Рисунок 2.3 — Диаграмма классов
  04-er.puml         → Рисунок 2.4 — Диаграмма «сущность–связь»

Как получить PNG/SVG для вставки в Word:

1. Откройте https://plantuml.com/
2. Скопируйте ВЕСЬ текст из нужного файла .puml (включая @startuml и @enduml)
3. Вставьте в левую панель редактора — диаграмма появится справа
4. Экспорт: правый клик по диаграмме → Save image / или кнопка экспорта
5. В Word: Вставка → Рисунок, подпись «Рисунок 2.X – …», источник: собственная разработка

Альтернатива в VS Code / Cursor:
  Расширение «PlantUML» + Java, либо команда:
  java -jar plantuml.jar -tpng diagrams/*.puml

Рекомендуемый масштаб для печати: ширина 14–16 см в пояснительной записке.

Версии Mermaid (mermaid.live): см. подпапку mermaid\
