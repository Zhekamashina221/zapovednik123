Диаграммы Mermaid для диплома «Zapovednik»
==========================================

Просмотр и экспорт в PNG/SVG:

  https://mermaid.live/

  1. Откройте сайт
  2. Скопируйте содержимое нужного файла .mmd
  3. Вставьте в редактор слева
  4. Actions → Export PNG или SVG

Файлы:

  01-use-case.mmd       — варианты использования (flowchart, т.к. в Mermaid нет Use Case)
  02-activity.mmd       — деятельность: карточка, избранное, маршрут
  02-activity-app.mmd   — деятельность: всё приложение
  03-class.mmd          — диаграмма классов
  04-er.mmd             — ER-диаграмма (SQLite)

В Cursor / VS Code:

  Расширение «Markdown Preview Mermaid Support» или встроенный предпросмотр .md
  с блоком:

    ```mermaid
    (вставить код из .mmd без первой строки %%{init...}%% при ошибке)
    ```

Примечание:

  Для официальной графической части (А1) нормоконтроль часто принимает PlantUML
  из папки ../ (01-use-case.puml и т.д.). Mermaid удобен для правок и mermaid.live.

  Диаграмма вариантов использования в Mermaid — схематичная (flowchart).
  Для классического вида «овал + человечек» лучше PlantUML.
