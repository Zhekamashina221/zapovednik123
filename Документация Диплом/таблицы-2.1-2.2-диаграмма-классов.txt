Атрибуты классов и операции классов представлены в таблицах 2.1 и 2.2.

Таблица 2.1 – Атрибуты классов

Наименование класса	Имя атрибута	Тип данных	Описание атрибута
User	
id	Integer	Идентификатор пользователя
email	String	Электронная почта (уникальная)
passwordHash	String	Хеш пароля
name	String	Имя пользователя
role	String	Роль (user / admin)
avatarUrl	String	URL аватара
isBlocked	Boolean	Признак блокировки учётной записи
emailVerified	Boolean	Подтверждение электронной почты

Reserve	
id	Integer	Идентификатор территории
name	String	Название ООПТ
type	String	Тип особо охраняемой территории
region	String	Область
district	String	Район
latitude	Real	Широта
longitude	Real	Долгота
area	Real	Площадь, га
description	String	Описание
isPublished	Boolean	Признак публикации на портале
phone	String	Контактный телефон
email	String	Контактный e-mail территории
website	String	Адрес веб-сайта

Photo	
id	Integer	Идентификатор фотографии
reserveId	Integer	Идентификатор ООПТ
url	String	URL изображения

Favorite	
id	Integer	Идентификатор записи
userId	Integer	Идентификатор пользователя
reserveId	Integer	Идентификатор ООПТ

Review	
id	Integer	Идентификатор отзыва
reserveId	Integer	Идентификатор ООПТ
userId	Integer	Идентификатор автора
rating	Integer	Оценка от 1 до 5
comment	String	Текст комментария
status	String	Статус модерации
createdAt	DateTime	Дата создания
updatedAt	DateTime	Дата изменения

CuratedRoute	
id	Integer	Идентификатор подборки
slug	String	URL-идентификатор подборки
title	String	Заголовок подборки
profile	String	Профиль маршрута
reserveIds	String	Список идентификаторов ООПТ
distanceM	Integer	Длина маршрута, м
durationS	Integer	Длительность, с
isPublished	Boolean	Признак публикации подборки

UserSavedRoute	
id	Integer	Идентификатор сохранённого маршрута
userId	Integer	Идентификатор пользователя
reserveId	Integer	Идентификатор ООПТ
profile	String	Профиль маршрутизации
title	String	Название маршрута
geojson	String	Геометрия маршрута (GeoJSON)
distanceM	Real	Длина маршрута, м
durationS	Real	Длительность, с
createdAt	DateTime	Дата сохранения

AuditLog	
id	Integer	Идентификатор записи журнала
userId	Integer	Идентификатор администратора
actionType	String	Тип действия
entityType	String	Тип сущности
entityId	Integer	Идентификатор сущности
createdAt	DateTime	Дата и время действия

Источник: собственная разработка

Таблица 2.2 – Операции классов

Наименование класса	Название операции	Описание операции
User	
authenticate()	Аутентификация по e-mail и паролю
register()	Регистрация нового пользователя
update()	Обновление профиля (имя, аватар)
change_password()	Смена пароля
has_role()	Проверка роли пользователя
block()	Блокировка учётной записи (администратор)

Reserve	
get_by_id()	Получение карточки ООПТ по идентификатору
get_all()	Получение каталога с фильтрацией
get_filters()	Получение справочников для фильтров
create()	Создание записи об ООПТ (администратор)
update()	Обновление данных ООПТ (администратор)
delete()	Удаление ООПТ (администратор)

Photo	
get_by_reserve()	Получение фотографий территории
add()	Добавление фотографии (администратор)
delete()	Удаление фотографии (администратор)

Favorite	
get_by_user()	Список избранных ООПТ пользователя
add()	Добавление ООПТ в избранное
remove()	Удаление из избранного

Review	
get_by_reserve()	Получение отзывов по территории
get_by_user()	Получение отзывов пользователя
create()	Создание отзыва
update()	Обновление отзыва
delete()	Удаление отзыва
moderate()	Изменение статуса модерации (администратор)

CuratedRoute	
get_all()	Получение списка подборок маршрутов
get_by_slug()	Получение подборки по slug
create()	Создание подборки (администратор)
update()	Обновление подборки (администратор)
delete()	Удаление подборки (администратор)

UserSavedRoute	
get_by_user()	Список сохранённых маршрутов пользователя
get_by_id()	Получение маршрута по идентификатору
create()	Сохранение маршрута
delete()	Удаление сохранённого маршрута

AuditLog	
log_action()	Фиксация действия администратора в журнале
get_all()	Получение записей журнала аудита

RoutingService	
build_route()	Построение маршрута через внешний сервис ORS

Источник: собственная разработка
